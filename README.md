# save-contacts-outlook

1. Create `%USERPROFILE%\save-contacts-outlook`

2. Save all files in that directory

3. Go into MS Outlook and press `ALT + F11`

4. Import the `.cls` file with right click on the left side. And save and close.

5. Outlook -> File -> Options -> Customize Ribbon

6. Add a new group at the right side and name it.

7. Choose Macros from dropdown and add `SendSelectedMailToPython`
